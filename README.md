# vue搭建的网站闲鸭源码（PC前端）
------------------------
#### 产品介绍
+ 闲鸭源码是一个提供在线免费下载源码项目和网页模板的网站，
+ 管理员可以选择在后台发布源码，编写项目文档，人员管理，订单管理等
#### 服务端源码地址
> 项目API接口-->Node版本：https://github.com/MetaEth/codetrade-api-Node

> 项目API接口-->Java版本：https://github.com/MetaEth/codetrade-api-Java

> 项目后台管理：https://github.com/MetaEth/codetrade-admin

#### 项目使用

``` bash
# Clone project

git clone https://github.com/MetaEth/codetrade-views.git

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test

```
